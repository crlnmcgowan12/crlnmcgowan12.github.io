Campus Life Hub - Project Overview

Project Name

Campus Life Hub

This project is a simple informational website designed to keep UNC Charlotte students connected to campus life. It offers easy access to upcoming events, club activities, and contact information for campus resources—all in one place.

Project Overview

Function/Purpose: The purpose of Campus Life Hub is to help students stay involved and updated on what’s happening around campus through an organized and user-friendly web interface.

Intended Users: UNC Charlotte students, both new and returning, who want quick access to campus events and resources.

Overview of Content: The site includes five main pages: Home, About, Events, Contact, and Project Overview. Each provides helpful information or interactivity related to campus engagement.

Dynamic Functionality

Description: The Events page will include an interactive JavaScript feature that displays event cards dynamically from a JSON file, allowing new events to be added without editing HTML directly.

Purpose: To make the site feel current and automatically updated as new campus events are added.
